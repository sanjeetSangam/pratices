## TITLE

YAML is a human-readable data serialization language. It is often used for configuration files, but it can also be used for data exchange between languages.

## SYNTAX

YAML is a whitespace-sensitive language. This means that the indentation of the code matters. YAML uses spaces, not tabs, for indentation.

## EXAMPLE

```yaml
name: John Doe
age: 30
occupation: Software Engineer
```
